# AI-Powered UK Market Intelligence System

## 🚀 Project Journey (End-to-End Development)

This project was built from scratch as part of a structured effort to develop a real-world, AI-driven business intelligence system.

The goal was not just to build a dashboard, but to design a system that simulates how AI can support strategic business decisions.

---

## 🎯 Objective

To create an intelligent system that:

- Converts raw business data into structured insights  
- Allows users to ask business-focused questions  
- Provides immediate, data-driven answers  
- Mimics AI-powered decision support systems  

---

## 🧩 Phase 1 — Data Foundation

The project began by designing a structured dataset using Excel.

Key steps:

- Built a structured Excel engine for business intelligence use cases  
- Created a large bank of business-related user questions  
- Expanded the question library to around *1000 possible user queries*  
- Organised the logic so questions map to insights and outputs  
- Designed the Excel file as the *core intelligence layer*  

This created a query-response system instead of a static dataset.

---

## 🧠 Phase 2 — Logic and Insight Design

The project moved beyond simple data storage into decision logic.

Key steps:

- Created structured question categories  
- Mapped questions to:
  - recommended actions  
  - confidence levels  
  - business impact  
  - use cases  
- Designed an AI-style reasoning flow using Excel sheets  

Sheets included:

- Insights engine  
- AI response templates  
- Decision rules  
- User query testing  

This transformed the Excel into a *rule-based AI-like engine*.

---

## 📊 Phase 3 — Business Intelligence Layer

The system was designed to answer real business questions such as:

- Which regions have strongest business activity?  
- Where should a business expand?  
- What areas show growth potential?  

This stage focused on turning data into *decision-making insights*.

---

## 💻 Phase 4 — Streamlit App Development

The Excel engine was integrated into a Python application.

Key steps:

- Built app.py using Python  
- Connected Excel engine using Pandas  
- Created an interactive UI using Streamlit  
- Enabled user input for business questions  
- Displayed structured outputs instantly  

This converted the system into an *interactive AI-style application*.

---

## ⚙️ Phase 5 — Debugging & Problem Solving

During development, several real-world issues were solved:

- Excel file size issues on GitHub  
- Incorrect file paths  
- File naming problems  
- Double extensions (.xlsx.xlsx)  
- Streamlit execution errors  
- Sheet name mismatches  
- Folder structure issues  

Fixing these reflects real engineering experience.
